knawd deployer

knawd deployer: Deploy a WASM enabled crun into kubernetes to enable running WASM libraries in kubernetes and knative. See https://knawd.dev for more information.

— Open in Artifact Hub

Install the Repo

helm repo add core-dump-handler https://knawd.github.io/deployer/

Install the Chart

helm install knawd-deployer --create-namespace --namespace knawd . -f TARGET.yaml
